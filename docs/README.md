# Meteora Web Project

## Overview
This is the completely new implementation for the Meteora website. It is built natively with HTML, CSS, and modern JavaScript, eliminating framework bloat while providing a highly responsive and modern feel.

## Structure
- `/assets/`: Images, icons, and logo assets.
- `/components/`: Reusable components and logic (e.g. `main.js`).
- `/sections/`: For modularizing HTML sections if needed later.
- `/styles/`: All site styles (`index.css` and `variables.css`).
- `/public/`: Public build assets.

## Design
- Theme: Clean, modern corporate white
- Primary Color: Deep Tech Blue
- Typography: Inter (Google Fonts)

## Features
- Fully responsive layout
- Micro-animations and reveal-on-scroll using IntersectionObserver
- Icon integration with Lucide
